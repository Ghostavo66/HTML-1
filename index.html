Tags de estrutura

Tags de estrutura são o que compõe o minimo para o funcionamento da página HTML, por exemplo:
<head>: onde ficam as informações da página, como título, charset, links para arquivos externos, etc.
<body>: onde fica o conteúdo visível da página, como textos, imagens, links, etc.
Todas as tags devem ser abertas e fechadas corretamente para garantir o funcionamento da página, utilizando o sinal de menor que (<) para abrir e maior que (>) para fechar.

Tags de títulos

As tags de títulos são utilizadas para definir a hierarquia dos títulos na página, indo do mais importante (h1) ao menos importante (h6).
Exemplo:
<h1>Título 1</h1>
<h2>Título 2</h2>
<h3>Título 3</h3>
(Voltar)

Tags de listas

As tags de listas são utilizadas para criar listas ordenadas (ol) e não ordenadas (ul), com itens representados pela tag li.
Exemplo de lista ordenada:
<ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
Exemplo de lista não ordenada:
<ul>
  <li>Item A</li>
  <li>Item B</li>
</ul>
(Voltar)

Tags de destaque

As tags de destaque são utilizadas para enfatizar partes do texto, como negrito (strong), itálico (em) e sublinhado (u).
Exemplo:
<strong>Texto em negrito</strong>
<i>Texto em itálico</i>
<u>Texto sublinhado</u>
